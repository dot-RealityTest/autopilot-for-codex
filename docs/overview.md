# Autopilot for Codex Overview

Autopilot for Codex is a local-first macOS menu bar utility for developers using Codex automations.

It watches automation configuration and memory files under `~/.codex/automations`, then shows a calm operational view of what is running, what is waiting for review, and what needs attention.

## Core Value

Autopilot for Codex helps developers understand local automation state quickly:

- which Codex automations are active,
- which automations need approval,
- what changes are proposed,
- which permissions may be needed,
- when a run happened or will happen next.

## Product Positioning

Autopilot for Codex is not a monitoring dashboard or enterprise reporting tool. It is a native macOS control layer for autonomous developer workflows.

The product is designed around calm visibility, trust, and lightweight human approval.

## Privacy

The app reads local Codex automation files and does not upload automation data.
